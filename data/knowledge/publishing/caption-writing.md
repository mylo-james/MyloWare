# Caption Writing Guide

Engagement-driving caption structures and formulas.

---

## Caption Purpose

### What Captions Do

| Function | How |
|----------|-----|
| **Hook extension** | Adds curiosity beyond visual |
| **Context** | Explains what viewer will see |
| **Engagement** | Prompts comments/shares |
| **SEO** | Searchable keywords |
| **Personality** | Shows brand voice |

### Caption Impact

- Captions are searchable (TikTok SEO)
- Can boost or clarify the hook
- Drive specific engagement actions
- Show personality and voice

## Caption Length

### Length Guidelines

| Length | Characters | Best For |
|--------|------------|----------|
| **Ultra-short** | < 50 | Quick jokes, memes |
| **Short** | 50-100 | Most content |
| **Medium** | 100-200 | Stories, tutorials |
| **Long** | 200-500 | Educational, detailed |

### Length by Content Type

| Content | Recommended |
|---------|-------------|
| ASMR/Satisfying | Short or minimal |
| Tutorial | Medium (explain steps) |
| Story | Medium to long |
| Comedy | Short and punchy |
| Educational | Medium to long |

## SEO Optimization

### Searchable Captions

TikTok search indexes caption text.

**Include**:
- Key topic words
- Niche terms people search
- Problem/solution language

**Example** (SEO-optimized):
```
ASMR glass cutting sounds for sleep and relaxation ðŸ˜´
The perfect tingles to help you unwind

#asmr #sleepsounds #relaxation #glasscutting
```

### Keyword Placement

- Put important keywords early
- Use natural language (not stuffed)
- Match what users would search

## Caption Mistakes

### What to Avoid

| Mistake | Why |
|---------|-----|
| "Link in bio" repeatedly | Feels salesy, algorithm may suppress |
| "Follow for part 2" | Engagement bait, can backfire |
| All caps throughout | Feels like shouting |
| No caption at all | Misses SEO opportunity |
| Essay-length every time | Most people won't read |
| Hashtags before caption | Looks cluttered |

### Better Alternatives

| Instead of | Try |
|------------|-----|
| "Follow for more" | (Just make good content) |
| "Like if you agree" | "Which one resonates with you?" |
| "Link in bio!!" | "DM 'guide' for the link" |
| No CTA | "Save this for later ðŸ“Œ" |
