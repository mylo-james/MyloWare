# Text Overlay Guide

Typography, timing, and placement for video text.

---

## Text Purpose

### When to Use Text

| Use Case | Purpose |
|----------|---------|
| **Hook text** | Reinforce opening hook |
| **Subtitles** | Accessibility, sound-off viewing |
| **Key points** | Emphasize important info |
| **Labels** | Identify objects/steps |
| **CTA** | Prompt action at end |

### When NOT to Use

- Cluttering visual content
- When audio is sufficient
- When visuals are self-explanatory
- In ASMR (minimal text preferred)

## Typography Basics

### Font Selection

| Context | Font Style | Examples |
|---------|------------|----------|
| **Bold/Impact** | Sans-serif, heavy | Impact, Bebas Neue, Montserrat Bold |
| **Readable** | Clean sans-serif | Helvetica, Arial, SF Pro |
| **Casual** | Rounded, friendly | Nunito, Comic Sans (sparingly) |
| **Elegant** | Serif or thin sans | Playfair Display, Lato Thin |

### Font Rules

| Rule | Why |
|------|-----|
| Max 2 font families | Consistency |
| Minimum 24pt for mobile | Readability |
| High contrast colors | Visibility |
| Consistent style | Professional look |

## Styling

### Text Contrast

Make text readable on any background:

| Technique | When to Use |
|-----------|-------------|
| **White + black shadow** | Most versatile |
| **Black + white stroke** | Light backgrounds |
| **Color + dark background box** | Maximum readability |
| **Outlined text** | Busy backgrounds |

### Shadow Settings

```
Good shadow for readability:
- Offset: 2-4px
- Blur: 2-4px
- Color: Black or dark version of text color
- Opacity: 50-80%
```

### Background Boxes

```
Semi-transparent background:
- Color: Black or matching theme
- Opacity: 40-70%
- Padding: 8-16px
- Rounded corners: 4-8px (optional)
```

## Positioning

### Safe Zones for Text

```
+---------------------------------+
|  [x] Top Zone (safe)             |  8-15% from top
+---------------------------------+
|                                 |
|       [x] Center (safest)        |  Middle 50%
|                                 |
+---------------------------------+
|  [!]  Bottom Zone               |  Avoid bottom 20%
|  (UI overlays here)            |  (TikTok captions)
+---------------------------------+
```

### Position by Text Type

| Text Type | Position |
|-----------|----------|
| Hook text | Center or upper third |
| Subtitles | Lower third (above UI) |
| Labels | Near the object |
| Key points | Center, one at a time |
| CTA | Center or lower third |

### Horizontal Alignment

| Alignment | Use For |
|-----------|---------|
| **Center** | Hook text, titles, emphasis |
| **Left** | Subtitles, lists, steps |
| **Right** | Rarely (feels unnatural) |

## Timing

### Text Duration

| Text Length | Duration |
|-------------|----------|
| 1-3 words | 1.5-2 seconds |
| 4-7 words | 2-3 seconds |
| 8+ words | 3-4 seconds |
| Subtitles | Match speech duration |

### Reading Speed Rule

> Average reading speed: 3-4 words per second
>
> Give viewers 1.5x the read time for comfortable comprehension

### Animation Timing

| Animation | Duration |
|-----------|----------|
| Fade in | 0.2-0.3s |
| Hold | (Based on word count) |
| Fade out | 0.2-0.3s |
| Cut (no animation) | Instant |

---

## Animation Styles

### Entry Animations

| Animation | Effect | Use For |
|-----------|--------|---------|
| **Fade** | Subtle, professional | General use |
| **Pop/Scale** | Energetic, attention | Emphasis |
| **Slide** | Dynamic, directional | Lists, steps |
| **Type on** | Character by character | Dramatic reveal |

### Exit Animations

| Animation | Effect |
|-----------|--------|
| **Fade out** | Clean, professional |
| **Cut** | Quick, no nonsense |
| **Slide out** | Matches slide in |
| **Scale down** | Energetic |

### When to Skip Animation

- Fast-paced content (just cut)
- ASMR/relaxing (minimal movement)
- When speed matters

## Subtitles/Captions

### Subtitle Best Practices

| Practice | Why |
|----------|-----|
| 1-2 lines max | Easy to read |
| Sync to speech | Matches audio |
| Position above UI | Stays visible |
| High contrast | Readable everywhere |

### Subtitle Styling

```
Recommended subtitle style:
- Font: Clean sans-serif (SF Pro, Helvetica)
- Size: 28-36pt
- Color: White
- Shadow: Black, 2px offset, 50% opacity
- Position: 15-20% from bottom
- Max width: 80% of frame
```

### Auto-Caption Tools

- TikTok built-in captions
- CapCut auto-captions
- Descript
- Rev

## Text for Different Content

### ASMR Content

- Minimal text
- Soft colors
- No jarring animations
- Subtitles optional (often distracting)

### Tutorial Content

- Step numbers
- Key terms highlighted
- Clear hierarchy
- Timer/progress if relevant

### Comedy Content

- Bold, impactful fonts
- Quick timing
- Punchy animation
- Supports the joke

### Educational Content

- Clear structure
- Bullet points
- Definitions highlighted
- Consistent style

## Common Mistakes

### Text Mistakes to Avoid

| Mistake | Problem | Fix |
|---------|---------|-----|
| Too much text | Overwhelming | One idea per screen |
| Too small | Unreadable on mobile | Min 24pt |
| No contrast | Can't read | Add shadow or background |
| Wrong position | Covered by UI | Stay in safe zones |
| Too fast | Can't read | Follow timing rules |
| Too slow | Boring | Match content pace |
| Too many fonts | Unprofessional | Max 2 fonts |

---

## Quick Reference

### Text Checklist

- [ ] Font is readable (sans-serif, 24pt+)
- [ ] High contrast (shadow or background)
- [ ] In safe zone (not covered by UI)
- [ ] Timing allows reading
- [ ] Animation matches content vibe
- [ ] Consistent style throughout
- [ ] Not cluttered

### Quick Settings

```
Default settings that usually work:

Font: Montserrat Bold or Helvetica Bold
Size: 32-40pt
Color: White
Shadow: Black, 2px, 50%
Position: Center, upper third
Duration: 2-3 seconds per text block
Animation: Fade or Pop
```
