# Archived Prompts

**THIS DIRECTORY IS NOT INGESTED INTO THE DATABASE**

This directory is OUTSIDE the `prompts/` directory, so the ingestion script never scans it.

Contents:
- `v2/` - Original v2 prompt development files
- `deleted-prompts/` - Old prompts that have been replaced

All active prompts are in:
- `prompts/personas/`
- `prompts/projects/`
- `prompts/combinations/`
- `prompts/memory/`
- `prompts/workflows/`

Use git history to track changes to active prompts, not version suffixes.
